EchoPrint
Android
Keras
TensorFlow
Acoustic Sensing
Python

An alternative face authentication method for smart-phones.
EchoPrint uses a combination of Acoustic signals (Sound waves) along with visual features (Image recognition) to create a fool-proof face authentication system.
The smartphone is made to emit a high frequecy sound wave that reflects on the person's face. This reflection is recorded and Deep learning on the reflection is used to classify the person as authorized/un-authorized person.
Achieved accuracy of 96% using Tensorflow and keras with ensembeled model of CNN and SVM.
CNN is used for feature extraction and SVMs is used for calssification over the acoustic and visual features.

Kym is the alternative needed for a better credit score calculation. An android and web based personal assistant, Kym captures data - financial, social and behavioural. *Kym automatically builds model(s) that help companies in assigning credit scores, insurances, etc.

*Built a data extractor to extract content from SMS and Emails

Enhanced version of GoPay using Kym. Lets users pay for services using GoPay wallet even without having money in wallet. The maximum amount of negative balance allowed is calculated by machine learning (Implemented Multiple linear regressions).

Social lending platform using Kym:
Built a social medial platform using Kym backend at YHacks 2017 (Yale University). The platform is like Yelp for social lending - Lenders and borrowers can discover each other along with credit ratings.
The platform helps lenders invest money into prospective borrowers in neighborhood with varying interest rates based on borrowers credit profile, which is built using Kym backend.
